# Python-If-Else-Statements---Conditional-Statements
In Python, If-Else is a fundamental conditional statement used for decision-making in programming.
If...Else statement allows to execution of specific blocks of code depending on the condition is True or False.
if Statement

if statement is the most simple decision-making statement. If the condition evaluates to True, the block of code inside the if statement is executed.

example:

i = 20

if i > 0:
    print("i is positive")
else:
    print("i is 0 or Negative")


If Else in One-line 

example:

res = "Positive" if a >= 0 else "Negative"
print(res)
